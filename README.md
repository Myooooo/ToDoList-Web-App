# TODO App

A browser-based todo application with:

- My Day / Important / All Tasks views
- Task steps, due dates, recurrence
- Search and filters
- Import/export data

## Project Structure

- `index.html` - App markup
- `assets/css/todo-app.css` - App styles
- `assets/js/todo-app.js` - App logic
- `docs/legacy-inline-backup.html` - Backup of previous inline version

## Run

Open `index.html` directly in a browser.
